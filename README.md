# Employee Salary & Promotion Analysis
📌 Project Overview
This project analyzes employee salary distribution and promotion patterns using SQL.
The goal is to understand salary differences across departments, identify promotion gaps, and detect employees who may be eligible for promotion.

The analysis is performed using SQL queries including aggregate functions, window functions, and subqueries.

🎯**Project Objectives**
- Analyze salary distribution across departments and job roles
- Identify promotion gaps and average time employees wait for promotion
- Detect employees eligible for promotion
- Compare employee salaries with department average salaries
- Identify top-paid employees in each department

🗂 **Dataset Information**

  Dataset: IBM HR Analytics Employee Attrition Dataset
  Source: Kaggle

The dataset contains employee information including:

  **Column**	                                  **Description**
  EmployeeNumber	                            Unique ID of employee
  Age	                                        Employee age
  Department	                                Department name
  JobRole	                                    Employee job role
  MonthlyIncome	                              Monthly salary
  PerformanceRating	                          Performance score
  YearsAtCompany	                            Total years in company
  YearsSinceLastPromotion	                    Years since last promotion

🛠 **Tools Used**
SQL (PostgreSQL / pgAdmin)
Excel (for basic visualization)
PowerPoint (for presentation)

🧹 Data Cleaning :

Before analysis, several data quality checks were performed:

Checked missing values
Verified duplicate employee records
Identified salary outliers
Standardized department and education fields
Validated promotion year values

📈 **Key Insights**
Research & Development department has the highest salary distribution.
Managers and Research Directors earn the highest average salaries.
Some employees have worked 5+ years without promotion, indicating promotion gaps.
Salary ranking highlights top earners within each department.
High performing employees earning below average salary may have salary growth potential.

📌 **Conclusion**
This project demonstrates how SQL can be used for HR analytics, including salary analysis, promotion tracking, and employee performance evaluation.
These insights help organizations improve promotion strategies, salary fairness, and workforce planning.
